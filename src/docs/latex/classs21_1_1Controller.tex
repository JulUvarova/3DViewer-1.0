\doxysubsection{Detailed Description}
The \doxylink{classs21_1_1Controller}{Controller} class provides an interface for manipulating and updating a 3D scene. 

The \doxylink{classs21_1_1Controller}{Controller} acts as a singleton facade that connects the user interface with the underlying 3D scene operations. It allows for loading scenes, applying transformations such as scaling, translation, and rotation, and setting update callbacks for when the scene changes.

\doxysubsection{Detailed Description}
The \doxylink{classViewport3D}{Viewport3D} class provides a 3D viewport widget using modern Open\+GL. 

This widget handles Open\+GL initialization, shader setup, and rendering of a 3D scene. It supports updating the scene, handling resizing, and toggling between different rendering modes.

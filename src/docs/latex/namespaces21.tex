\doxysubsection{Detailed Description}
Contains vector and matrix structures and operations for 2D, 3D, and 4D computations. 

Contains classes and structures for parsing and managing OBJ file data.

Contains utilities for building transformation matrices and related data structures.

This namespace provides implementations of 2D, 3D, and 4D vectors, as well as a 4x4 matrix, commonly used in computer graphics, game development, and scientific computing.

\doxysubsection{Detailed Description}
Represents the indices for a vertex in a face, including vertex, texture coordinate, and normal indices. 

This structure holds the indices referencing the vertices, texture coordinates, and normals defined in the OBJ file. An index of -\/1 indicates that the corresponding data is not present.

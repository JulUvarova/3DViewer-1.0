\doxysubsection{Detailed Description}
Provides functionality to read and process OBJ files. 

The {\ttfamily \doxylink{classs21_1_1FileReader}{File\+Reader}} class offers a simple interface for reading OBJ files. It encapsulates the logic for parsing the file, normalizing the data, and returning the processed information as an {\ttfamily \doxylink{classs21_1_1OBJData}{OBJData}} object.

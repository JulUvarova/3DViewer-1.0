\doxysubsection{Detailed Description}
Singleton class that provides a unified interface for scene management. 

The \doxylink{classs21_1_1Facade}{Facade} class serves as a central point of interaction between the user and the underlying components responsible for file reading, scene data management, and transformation parameters. It employs the singleton pattern to ensure that only one instance exists throughout the application\textquotesingle{}s lifetime.

\begin{DoxyNote}{Note}
This class is non-\/copyable and non-\/movable to enforce the singleton design. 
\end{DoxyNote}

Detailed Description

A utility class for creating 4x4 transformation matrices.

This class provides static methods to generate 4x4 matrices for rotation, translation, and scaling transformations in 3D space. It is designed as a static utility class and cannot be instantiated.

Note
All methods are static, and the constructor is private to prevent instantiation.

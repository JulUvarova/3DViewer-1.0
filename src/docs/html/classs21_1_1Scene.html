Detailed Description

Manages the loading and transformation of mesh data for a 3D scene.

The Scene class is designed to handle mesh data operations, such as loading from an OBJData object and applying transformations to the mesh vertices. It maintains a private vector of vertices in homogeneous coordinates and a shared pointer to rendering data.
